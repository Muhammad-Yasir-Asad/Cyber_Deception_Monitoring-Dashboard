🆘 COMMON ISSUES

Issue: Port 5000 in use
Fix:   $ lsof -i :5000 | grep node | awk '{print $2}' | xargs kill -9

Issue: npm install fails
Fix:   $ rm -rf node_modules package-lock.json && npm install

Issue: "Cannot find module 'socket.io'"
Fix:   $ cd server && npm install socket.io

Issue: Dashboard won't connect to server
Fix:   • Check server running: http://localhost:5000/api/events
       • Check client .env has: REACT_APP_API_URL=http://localhost:5000

Issue: No data showing
Fix:   • Mock data loads automatically
       • Check browser console for errors
       • Verify npm install completed
